Conclusion

Based on 3-seed cross-validation (seeds: 2026, 2027, 2028) using Hit Rate@10, NDCG@10, and Precision@10, the Hybrid Recommendation System achieves the highest Accuracy score (HR×0.50 + NDCG×0.30 + Precision×0.20), outperforming all individual algorithms.

The Hybrid approach combines the strengths of:

  • Content-Based Filtering: Good for new items and users with specific preferences
  • User-Based CF: Effective at discovering serendipitous recommendations from similar users
  • Item-Based CF: Great for finding similar products based on user behavior patterns
  • SVD: Excellent at handling sparse data and discovering latent features

By combining these approaches with optimized weights, the Hybrid system achieves better overall performance, higher user satisfaction, and more diverse recommendations.
